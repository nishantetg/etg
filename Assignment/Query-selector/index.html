# Query Selector Query Selector All
1 querySelector() is jQuery functions which helps the HTML elements to be passed as a parameter by using CSS selectors (‘id’, ‘class’) can be selected. querySelectorAll() is jQuery functions which helps the HTML elements to be passed as a parameter by using CSS selectors (‘id’, ‘class’) can be selected.
2 querySelectorAll() method returns all the elements within the document which matches the specified CSS selector(s) querySelector() method returns only the first occurance of the elements within the document.
3 It returns all the elements that matches with the selector in the form of a static NodeList object which is a collection of nodes. If multiple elements occurs, then it returns the result for only the first matching element.
4 To access each element, we usually use a loop Loop is not needed because there is only one element.
